# College Website

A responsive college website developed to manage and display college-related information using Excel sheets as the data source. The project provides a simple and user-friendly interface for accessing academic and administrative information.

## Features

* 📋 Display student information
* 👨‍🏫 Faculty details
* 📅 Timetable management
* 📢 College notices and announcements
* 📊 Data fetched from Excel sheets
* 📱 Responsive design for desktop and mobile

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Excel Sheets (Data Source)

## Project Structure

```
College-Website/
│── indexfiles.html
│── css/
│── js/
│── assets/
│── data/
│    └── *.xlsx
│── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Launch `index.html` in your web browser.
4. Make sure the required Excel files are available in the `data` folder.

## Purpose

This project was developed to demonstrate how Excel sheets can be used as a simple data source for a college website. It showcases frontend web development skills along with basic data integration.
